# Tweet-Deleter-Chrome-extension-

THIS IS COMPLETELY FREE, NO PAYWALLS, NO LIMITS.  
This extension can be used to delete tweets, replies and to undo retweets.  
WORKS ONLY FOR CHROME BROWSER.

## Steps to Use

1. Download the zip file and unzip it
2. Open Chrome and go to `chrome://extensions/`
3. Enable **Developer mode** (top right corner)
4. Select **Load unpacked** and choose the previously unzipped folder
5. Open X → go to your profile page → start using the extension

## Note

Twitter saves some old tweets in the archive tier.  
If deletion stops working:

- Stop the extension
- Comment, like, and retweet a couple of posts
- Close the tab
- Open a new tab
- Open your Twitter profile again
- Restart the extension
