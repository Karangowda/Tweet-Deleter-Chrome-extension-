# Tweet-Deleter-Chrome-extension-
THIS IS COMPLETELY FREE, NO PAYWALLS, NO LIMITS.
This extension can be used to delete tweets, replies and to undo retweets
WORKS ONLY FOR CHROME BROWSER

STEP 1 -- Download the zip file and unzip it 
STEP 2 -- open chrome and go to chrome://extensions/
STEP 3 --  now enable developer mode (top right hand corner) 
STEP 4 -- Select load unpacked and select the previousy unzipped file
STEP 5 -- open x and then your profile page and then start to use the extension.

NOTE- twitter saves some of the old tweets in archive tier, hence old tweets might take time to load 
if the extension stops to delete , please stop it comment like and retweet a couple of posts and then close the tab
open a new tab, open your twitter profile and use the extension.
